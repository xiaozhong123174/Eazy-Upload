# Eazy-Upload

通用上传组件，可以上传图片、视频、word、excel、ppt、mysql等文件，并能实时预览，组件样式后续陆续更新。


# 初始化安装
## 1.安装前端
> 进入 \Eazy-Upload\web 目录执行
```js
npm i
```
